## Overview

This project explores a mobile-first daily lesson flow for speech therapy focused on rhotacism (improving the "R" sound).

The experience is inspired by Duolingo’s lesson progression system, but redesigned for a healthcare context where emotional safety and confidence-building are important.

The prototype includes:

- Interactive lesson cards
- Progress tracking
- Smooth transitions and animations
- Supportive pronunciation feedback
- XP and streak completion rewards
- Responsive mobile-first design
- PWA support

---

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- vite-plugin-pwa

---

## Lesson Cards

The prototype contains 4 exercise types:

1. Pronunciation recording card
2. Listening comparison card
3. Syllable repetition practice
4. Mirror mode posture practice

---

## Innovation

Instead of aggressive right/wrong gamification, the experience focuses on confidence-based reinforcement.

Speech therapy can feel emotionally vulnerable, so the interface uses supportive feedback and calmer interactions to encourage consistency rather than punishment.
